Console Tic tac toe
===================
This is a command line version of the unbeatable ruby tic tac toe before I packaged the core game logic up into a gem.

Usage:
------
- To run the app:
  + Clone the repo.
  + cd to the directory.
  + Type $ bin/ttt
  + Play the game.
- To run the specs:
 + cd to the directory.
 + Type $ rspec
